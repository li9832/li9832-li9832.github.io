打包部署好的文件

tar -zcpPvf filename.tar.gz --exclude='*.log'  file1 file2 ...
# -z, --gzip, --gunzip, --ungzip   通过 gzip 过滤归档
# -c, --create               创建一个新归档
# -p, --preserve-permissions, --same-permissions 解/压文件权限信息
# -P, --absolute-names       不要从文件名中清除引导符‘/’（绝对路径 
# -v, --verbose              详细地列出处理的文件
# -f, --file=ARCHIVE         使用归档文件或 ARCHIVE 设备
# --exclude 				 排除特定的文件或文件夹

tar快速部署

将打包好的tar上传到新机器，执行cd / && tar -zxvf ~/filename.tar.gz解压文件。
